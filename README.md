clippy
======

Basic Clipboard Script.

Install
=======

No support yet. Just download and place file where you'd like. 
Just make sure it's in your path!


Usage
=====

$ ls -l | clippy         # Copies contents to clippy
$ clippy                 # Pastes

$ clippy Stuff to copy   # Copies contents to clippy
$ clippy                 # Pastes
